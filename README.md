# Python implementation of yuma-consensus with visualization tools

Sake sure to modify the `weights.json`

Structure

```json
{
    "0,1000": {
      "uids": [5, 6, 7, 8, 9],
      "weights": [1, 2, 3, 4, 5]
    }
}
```

Represents

- Uid: 0
- Stake: 1000
- Voted on uids: [5, 6, 7, 8, 9]
- Voted on weights: [1, 2, 3, 4, 5]
